# CPD model of the conterminous United States（continental） (ESI framework)
This repository contains the Curie Point Depth (CPD) model described in [Chen, C. et al., 2025, submitted to *Geophysical Research Letters*].

## Citation
If you use these data, please cite:
> Chen, C., Wang, Z., Gao, Y., & Zhu, Z. (2025). A 0.36° Curie Point Depth Model of the Conterminous United States Derived from a Prior-Constrained Equivalent Source Inversion Manuscript submitted to Geophysical Research Letters.
